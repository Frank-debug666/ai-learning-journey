# 1. 神经网络搭建整体流程

```
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader
```

一个标准神经网络通常包含以下步骤：

```
1. 准备数据 Dataset / DataLoader
2. 定义模型 nn.Module
3. 选择网络层 Linear / Conv / RNN / Transformer
4. 选择激活函数 ReLU / Sigmoid / Tanh / GELU
5. 选择损失函数 CrossEntropyLoss / MSELoss
6. 选择优化器 SGD / Adam / AdamW
7. 前向传播 forward
8. 计算损失 loss
9. 反向传播 backward
10. 参数更新 optimizer.step()
11. 验证模型效果
12. 保存和加载模型
```

------

# 2. 数据准备模块

## 2.1 Dataset

`Dataset` 用于定义数据如何被读取。

```
from torch.utils.data import Dataset

class MyDataset(Dataset):
    def __init__(self, data, labels):
        self.data = data
        self.labels = labels

    def __len__(self):
        return len(self.data)

    def __getitem__(self, index):
        x = self.data[index]
        y = self.labels[index]
        return x, y
```

------

## API：`torch.utils.data.Dataset`

### 作用

用于封装数据集，告诉 PyTorch：

```
数据有多少条？
每次取第 i 条数据时返回什么？
```

------

### 必须实现的方法

| 方法                 | 输入     | 输出              | 作用                    |
| -------------------- | -------- | ----------------- | ----------------------- |
| `__len__()`          | 无       | 数据集长度        | 让 PyTorch 知道数据总量 |
| `__getitem__(index)` | 样本索引 | 单个样本 `(x, y)` | 让模型按索引读取数据    |

------

### 输入分析

```
def __getitem__(self, index):
```

| 参数    | 含义           | 影响             |
| ------- | -------------- | ---------------- |
| `index` | 当前样本的索引 | 决定取哪一条数据 |

------

### 输出分析

```
return x, y
```

| 输出 | 含义     | 作用         |
| ---- | -------- | ------------ |
| `x`  | 输入特征 | 传入神经网络 |
| `y`  | 标签     | 用于计算损失 |

------

## 2.2 DataLoader

`DataLoader` 用于批量加载数据。

```
train_loader = DataLoader(
    dataset=train_dataset,
    batch_size=32,
    shuffle=True
)
```

------

## API：`torch.utils.data.DataLoader`

### 作用

把 Dataset 包装成可迭代对象，自动完成：

```
批量读取
打乱数据
多线程加载
组成 batch
```

------

### 常用参数

```
DataLoader(
    dataset,
    batch_size=32,
    shuffle=True,
    num_workers=0,
    drop_last=False
)
```

| 参数          | 是否必须 | 含义                          | 影响                           |
| ------------- | -------- | ----------------------------- | ------------------------------ |
| `dataset`     | 是       | 数据集对象                    | 决定加载哪些数据               |
| `batch_size`  | 否       | 每批样本数量                  | 越大训练越稳定，但显存消耗越大 |
| `shuffle`     | 否       | 是否打乱数据                  | 训练集一般设为 `True`          |
| `num_workers` | 否       | 加载数据的进程数              | 越大读取越快，但占用 CPU       |
| `drop_last`   | 否       | 是否丢弃最后不足 batch 的数据 | 对 BatchNorm 训练可能有帮助    |

------

### 输出分析

```
for x, y in train_loader:
    ...
```

| 输出 | 形状示例                 | 作用     |
| ---- | ------------------------ | -------- |
| `x`  | `[batch_size, features]` | 输入模型 |
| `y`  | `[batch_size]`           | 计算损失 |

------

# 3. 模型定义模块

## 3.1 基础模型结构

```
class NeuralNetwork(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Linear(256, 10)
        )

    def forward(self, x):
        return self.net(x)
```

------

## API：`torch.nn.Module`

### 作用

`nn.Module` 是 PyTorch 中所有神经网络模型的基类。

所有自定义模型都应该继承它。

------

### 基本结构

```
class Model(nn.Module):
    def __init__(self):
        super().__init__()
        # 定义网络层

    def forward(self, x):
        # 定义前向传播
        return output
```

------

### 输入分析

| 部分                 | 作用                   |
| -------------------- | ---------------------- |
| `__init__()`         | 定义网络结构           |
| `forward(x)`         | 定义数据如何经过网络   |
| `super().__init__()` | 初始化父类功能，必须写 |

------

### 输出分析

```
return output
```

| 输出     | 含义         | 作用                 |
| -------- | ------------ | -------------------- |
| `output` | 模型预测结果 | 传给损失函数计算误差 |

------

# 4. 网络层模块

------

# 4.1 全连接层 Linear

```
layer = nn.Linear(in_features=784, out_features=256)
```

------

## API：`torch.nn.Linear`

### 作用

线性变换层，也叫全连接层。

数学形式：

```
y = xW^T + b
```

------

### 参数分析

```
nn.Linear(
    in_features,
    out_features,
    bias=True
)
```

| 参数           | 是否必须 | 含义         | 影响                       |
| -------------- | -------- | ------------ | -------------------------- |
| `in_features`  | 是       | 输入特征维度 | 必须和输入数据最后一维一致 |
| `out_features` | 是       | 输出特征维度 | 决定该层神经元数量         |
| `bias`         | 否       | 是否使用偏置 | `True` 时表达能力更强      |

------

### 输入分析

```
x = torch.randn(32, 784)
output = layer(x)
```

| 输入 | 形状                        | 含义         |
| ---- | --------------------------- | ------------ |
| `x`  | `[batch_size, in_features]` | 一批输入特征 |

------

### 输出分析

| 输出     | 形状                         | 作用                     |
| -------- | ---------------------------- | ------------------------ |
| `output` | `[batch_size, out_features]` | 传入下一层或作为预测结果 |

------

# 4.2 卷积层 Conv2d

```
conv = nn.Conv2d(
    in_channels=3,
    out_channels=16,
    kernel_size=3,
    stride=1,
    padding=1
)
```

------

## API：`torch.nn.Conv2d`

### 作用

用于处理图像数据，提取局部空间特征。

------

### 参数分析

```
nn.Conv2d(
    in_channels,
    out_channels,
    kernel_size,
    stride=1,
    padding=0,
    bias=True
)
```

| 参数           | 是否必须 | 含义         | 影响                     |
| -------------- | -------- | ------------ | ------------------------ |
| `in_channels`  | 是       | 输入通道数   | RGB 图像为 3，灰度图为 1 |
| `out_channels` | 是       | 卷积核数量   | 决定输出特征图数量       |
| `kernel_size`  | 是       | 卷积核大小   | 越大感受野越大，参数更多 |
| `stride`       | 否       | 滑动步长     | 越大输出尺寸越小         |
| `padding`      | 否       | 边缘填充     | 可保持输出尺寸           |
| `bias`         | 否       | 是否使用偏置 | 通常默认 `True`          |

------

### 输入分析

```
x = torch.randn(32, 3, 224, 224)
output = conv(x)
```

| 输入 | 形状                                    | 含义     |
| ---- | --------------------------------------- | -------- |
| `x`  | `[batch_size, channels, height, width]` | 一批图像 |

------

### 输出分析

| 输出     | 形状                                                | 作用             |
| -------- | --------------------------------------------------- | ---------------- |
| `output` | `[batch_size, out_channels, new_height, new_width]` | 提取后的图像特征 |

------

# 4.3 池化层 MaxPool2d

```
pool = nn.MaxPool2d(kernel_size=2, stride=2)
```

------

## API：`torch.nn.MaxPool2d`

### 作用

对特征图进行下采样，减少计算量，保留主要特征。

------

### 参数分析

```
nn.MaxPool2d(
    kernel_size,
    stride=None,
    padding=0
)
```

| 参数          | 是否必须 | 含义         | 影响                   |
| ------------- | -------- | ------------ | ---------------------- |
| `kernel_size` | 是       | 池化窗口大小 | 决定每次压缩的区域     |
| `stride`      | 否       | 滑动步长     | 通常等于 `kernel_size` |
| `padding`     | 否       | 填充大小     | 影响输出尺寸           |

------

### 输入输出分析

```
x = torch.randn(32, 16, 112, 112)
output = pool(x)
```

| 输入形状             | 输出形状示例       | 作用           |
| -------------------- | ------------------ | -------------- |
| `[32, 16, 112, 112]` | `[32, 16, 56, 56]` | 降低特征图尺寸 |

------

```

```

------

## API：`torch.nn.Flatten`

### 作用

把多维特征展开成一维特征，常用于卷积层连接全连接层之前。

------

### 参数分析

```
nn.Flatten(

------

### 输入输出分析

```

```

| 输入形状         | 输出形状    | 作用                     |

| `[32, 16, 7, 7]` | `[32, 784]` | 转成全连接层可接收的格式 |

# 5. 激活函数模块

激活函数的作用是引入非线性能力。

如果没有激活函数，多层神经网络本质上仍然只是线性模型。

------

# 5.1 ReLU

```
activation = nn.ReLU()
```

------

## API：`torch.nn.ReLU`

### 数学公式

```
ReLU(x) = max(0, x)
```

------

### 作用

最常用的激活函数，用于隐藏层。

------

### 参数分析

```
nn.ReLU(inplace=False)
```

| 参数      | 含义             | 影响                                |
| --------- | ---------------- | ----------------------------------- |
| `inplace` | 是否原地修改输入 | `True` 可省显存，但可能影响梯度计算 |

------

### 输入输出分析

| 输入        | 输出              | 作用                     |
| ----------- | ----------------- | ------------------------ |
| 任意 Tensor | 小于 0 的值变成 0 | 引入非线性，缓解梯度消失 |

------

### 适用场景

```
隐藏层
CNN
MLP
大多数普通深度网络
```

------

# 5.2 Sigmoid

```
activation = nn.Sigmoid()
```

------

## API：`torch.nn.Sigmoid`

### 数学公式

```
Sigmoid(x) = 1 / (1 + e^(-x))
```

------

### 输出范围

```
0 到 1
```

------

### 输入输出分析

| 输入        | 输出                  | 作用               |
| ----------- | --------------------- | ------------------ |
| 任意 Tensor | `[0, 1]` 之间的概率值 | 常用于二分类输出层 |

------

### 注意事项

```
隐藏层中不推荐大量使用，容易梯度消失。
二分类任务中，如果使用 BCEWithLogitsLoss，则不要手动加 Sigmoid。
```

------

# 5.3 Tanh

```
activation = nn.Tanh()
```

------

## API：`torch.nn.Tanh`

### 数学公式

```
Tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
```

------

### 输出范围

```
-1 到 1
```

------

### 输入输出分析

| 输入        | 输出      | 作用                            |
| ----------- | --------- | ------------------------------- |
| 任意 Tensor | `[-1, 1]` | 常用于 RNN 或需要正负输出的场景 |

------

### 特点

```
比 Sigmoid 更适合隐藏层，但仍然可能梯度消失。
```

------

# 5.4 Softmax

```
activation = nn.Softmax(dim=1)
```

------

## API：`torch.nn.Softmax`

### 作用

把一组数转换成概率分布。

常用于多分类任务的最终概率输出。

------

### 参数分析

```
nn.Softmax(dim)
```

| 参数  | 是否必须 | 含义               | 影响                   |
| ----- | -------- | ------------------ | ---------------------- |
| `dim` | 是       | 沿哪个维度做归一化 | 分类任务通常为 `dim=1` |

------

### 输入输出分析

```
x = torch.randn(32, 10)
output = nn.Softmax(dim=1)(x)
```

| 输入形状                    | 输出形状                    | 作用               |
| --------------------------- | --------------------------- | ------------------ |
| `[batch_size, num_classes]` | `[batch_size, num_classes]` | 每一类对应一个概率 |

------

### 注意事项

```
如果使用 CrossEntropyLoss，不需要手动加 Softmax。
因为 CrossEntropyLoss 内部已经包含 LogSoftmax。
```

------

activation = nn.GELU()

## API：`torch.nn.GELU`

### 作用

------

### 参数分析

nn.GELU(approximate='none')
```

| 参数          | 含义             | 影响                               |
| ------------- | ---------------- | ---------------------------------- |

------

### 输入输出分析

# 6. 损失函数模块

损失函数用于衡量：

```
模型预测值 和 真实标签 之间的差距
```

训练目标就是让损失函数越来越小。

------

# 6.1 CrossEntropyLoss

```
criterion = nn.CrossEntropyLoss()
```

------

## API：`torch.nn.CrossEntropyLoss`

### 作用

用于多分类任务。

例如：

```
手写数字识别：0 到 9
图像分类：猫、狗、车、人
文本分类：正面、负面、中性
```

------

### 输入要求

```
loss = criterion(outputs, labels)
```

| 输入      | 形状                        | 含义              |
| --------- | --------------------------- | ----------------- |
| `outputs` | `[batch_size, num_classes]` | 模型输出的 logits |
| `labels`  | `[batch_size]`              | 类别索引          |

------

### 参数分析

```
nn.CrossEntropyLoss(
    weight=None,
    ignore_index=-100,
    reduction='mean'
)
```

| 参数           | 含义         | 影响                                         |
| -------------- | ------------ | -------------------------------------------- |
| `weight`       | 类别权重     | 处理类别不均衡                               |
| `ignore_index` | 忽略某个标签 | 常用于 NLP padding                           |
| `reduction`    | 损失汇总方式 | `'mean'` 平均，`'sum'` 求和，`'none'` 不汇总 |

------

### 输出分析

| 输出   | 含义        | 作用         |
| ------ | ----------- | ------------ |
| `loss` | 标量 Tensor | 用于反向传播 |

------

### 重要注意

```
# 正确
criterion = nn.CrossEntropyLoss()
outputs = model(x)
loss = criterion(outputs, labels)

# 错误
outputs = nn.Softmax(dim=1)(model(x))
loss = criterion(outputs, labels)
```

原因：

```
CrossEntropyLoss 内部已经包含 Softmax 相关计算。
模型最后一层直接输出 logits 即可。
```

------

# 6.2 MSELoss

```
criterion = nn.MSELoss()
```

------

## API：`torch.nn.MSELoss`

### 作用

用于回归任务。

例如：

```
房价预测
温度预测
股票数值预测
图像重建
```

------

### 输入要求

```
loss = criterion(pred, target)
```

| 输入     | 形状                       | 含义       |
| -------- | -------------------------- | ---------- |
| `pred`   | `[batch_size, output_dim]` | 模型预测值 |
| `target` | `[batch_size, output_dim]` | 真实数值   |

------

### 参数分析

```
nn.MSELoss(reduction='mean')
```

| 参数        | 含义         | 影响                      |
| ----------- | ------------ | ------------------------- |
| `reduction` | 损失汇总方式 | `'mean'` 常用，训练更稳定 |

------

### 输出分析

| 输出   | 含义                     | 作用             |
| ------ | ------------------------ | ---------------- |
| `loss` | 预测值和真实值的均方误差 | 用于回归模型优化 |

------

# 6.3 L1Loss

```
criterion = nn.L1Loss()
```

------

## API：`torch.nn.L1Loss`

### 作用

计算预测值与真实值之间的平均绝对误差。

------

### 输入输出分析

| 输入           | 输出      | 作用                  |
| -------------- | --------- | --------------------- |
| `pred, target` | 标量 loss | 对异常值不如 MSE 敏感 |

------

### 适用场景

```
回归任务
图像恢复
异常值较多的数据
```

------

# 6.4 BCEWithLogitsLoss

```
criterion = nn.BCEWithLogitsLoss()
```

------

## API：`torch.nn.BCEWithLogitsLoss`

### 作用

用于二分类或多标签分类。

它内部已经包含：

```
Sigmoid + Binary Cross Entropy
```

------

### 输入要求

```
loss = criterion(outputs, labels.float())
```

| 输入      | 形状                                         | 含义                     |
| --------- | -------------------------------------------- | ------------------------ |
| `outputs` | `[batch_size]` 或 `[batch_size, num_labels]` | 未经过 Sigmoid 的 logits |
| `labels`  | 相同形状                                     | 标签，值为 0 或 1        |

------

### 参数分析

```
nn.BCEWithLogitsLoss(
    weight=None,
    pos_weight=None,
    reduction='mean'
)
```

| 参数         | 含义           | 影响               |
| ------------ | -------------- | ------------------ |
| `weight`     | 每个样本的权重 | 调整样本重要性     |
| `pos_weight` | 正样本权重     | 处理正负样本不平衡 |
| `reduction`  | 汇总方式       | 控制 loss 输出形式 |

------

### 注意事项

```
# 正确
outputs = model(x)
loss = criterion(outputs, labels.float())

# 错误
outputs = torch.sigmoid(model(x))
loss = criterion(outputs, labels.float())
```

原因：

```
BCEWithLogitsLoss 内部已经包含 Sigmoid。
```

# 7. 优化器模块

优化器用于根据梯度更新模型参数。

------

# 7.1 SGD

```
optimizer = optim.SGD(
    model.parameters(),
    lr=0.01,
    momentum=0.9
)
```

------

## API：`torch.optim.SGD`

### 参数分析

------

## API：`optimizer.zero_grad()`

### 作用

清空上一轮的梯度。

------

### 为什么必须写？

PyTorch 的梯度默认会累加。

如果不清空梯度：

```
当前 batch 的梯度 = 当前梯度 + 之前 batch 的梯度
```

这通常会导致训练错误。

import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader

class MLP(nn.Module):
    def __init__(self):
        super().__init__()

        self.net = nn.Sequential(
            nn.Flatten(),
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Linear(128, 10)
        )

    def forward(self, x):
        return self.net(x)


model = MLP()

criterion = nn.CrossEntropyLoss()

optimizer = optim.Adam(
    model.parameters(),
    lr=0.001
)

num_epochs = 10

for epoch in range(num_epochs):
    model.train()

    for x, y in train_loader:
        optimizer.zero_grad()

        outputs = model(x)

        loss = criterion(outputs, y)

        loss.backward()

        optimizer.step()

    print(f"Epoch [{epoch+1}/{num_epochs}], Loss: {loss.item():.4f}")

# 15. 常见错误总结

------

## 错误 1：CrossEntropyLoss 前加 Softmax

```
# 错误
outputs = torch.softmax(model(x), dim=1)
loss = nn.CrossEntropyLoss()(outputs, y)
```

正确写法：

```
outputs = model(x)
loss = nn.CrossEntropyLoss()(outputs, y)
```

------

## 错误 2：BCEWithLogitsLoss 前加 Sigmoid

```
# 错误
outputs = torch.sigmoid(model(x))
loss = nn.BCEWithLogitsLoss()(outputs, y)
```

正确写法：

```
outputs = model(x)
loss = nn.BCEWithLogitsLoss()(outputs, y.float())
```

------

## 错误 3：忘记清空梯度

```
# 错误
loss.backward()
optimizer.step()
```

正确写法：

```
optimizer.zero_grad()
loss.backward()
optimizer.step()
```

------

## 错误 4：验证时忘记 `model.eval()`

```
# 错误
with torch.no_grad():
    outputs = model(x)
```

正确写法：

```
model.eval()

with torch.no_grad():
    outputs = model(x)
```

## 错误 5：验证时忘记 `torch.no_grad()`

```
# 错误
model.eval()
outputs = model(x)
```

正确写法：

```
model.eval()

# 20. 一句话总结

神经网络搭建的本质流程是：

```
用 DataLoader 送入数据，
用 nn.Module 定义模型，

用损失函数衡量预测错误，
用优化器根据梯度更新参数，
最后通过验证集评估模型效果。
