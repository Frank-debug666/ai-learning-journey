# 深度学习常见激活函数汇总

| **类别**   | **激活函数** | **PyTorch**           | **适用场景**                           | **核心思想**                                              | **优点**                                         | **缺点**                                             |
| ---------- | ------------ | --------------------- | -------------------------------------- | --------------------------------------------------------- | ------------------------------------------------ | ---------------------------------------------------- |
| 经典饱和   | Sigmoid      | `nn.Sigmoid()`        | 二分类输出层、LSTM/GRU门控单元         | 将输入压缩至 $(0, 1)$ 之间                                | 输出映射在固定区间，适合表示概率                 | 存在梯度消失问题；非零中心化（Non-zero centered）    |
| 经典饱和   | Tanh         | `nn.Tanh()`           | 传统RNN隐藏层、GAN的生成器输出层       | 将输入压缩至 $(-1, 1)$ 之间                               | 零中心化（Zero-centered），收敛速度快于 Sigmoid  | 依然存在梯度消失问题                                 |
| 线性截断   | ReLU         | `nn.ReLU()`           | 经典CNN、MLP等大多数网络的隐藏层       | $$f(x) = \max(0, x)$$ 负区间直接截断为0                   | 计算极快（仅需阈值判断）；缓解梯度消失           | Dead ReLU问题（部分神经元参数无法更新而“坏死”）      |
| 线性截断   | LeakyReLU    | `nn.LeakyReLU()`      | 解决Dead ReLU的隐藏层、GAN判别器       | $$f(x) = \max(\alpha x, x)$$ 负区间给予微小斜率 $\alpha$  | 解决了 Dead ReLU 问题，使所有神经元都能保持更新  | 斜率 $\alpha$ 属于超参数，需要人工调优               |
| 现代大模型 | GELU         | `nn.GELU()`           | Transformer、BERT、GPT等大模型隐藏层   | 引入随机正则思想，结合了ReLU与Dropout，按概率决定是否激活 | 在NLP和大型Transformer架构中表现极佳，泛化能力强 | 计算量相比 ReLU 略大                                 |
| 现代CV模型 | SiLU (Swish) | `nn.SiLU()`           | YOLOv5/v7/v8、EfficientNet等现代CV模型 | $$f(x) = x \cdot \sigma(x)$$ 无上界有下界、平滑且非单调   | 在深层网络中表现通常优于 ReLU，边缘更加平滑      | 计算稍慢（包含 Sigmoid 乘法）                        |
| 多分类输出 | Softmax      | `nn.Softmax(dim=...)` | 多分类任务的输出层                     | 归一化指数函数，将输出转化为标准概率分布（和为1）         | 能够直观输出各类别的概率，完美契合交叉熵损失     | 极值情况下容易出现数值上溢/下溢（PyTorch内部已优化） |

# 按任务/网络位置选择激活函数

| **任务或网络位置**           | **推荐激活函数**                 | **备注**                             |
| ---------------------------- | -------------------------------- | ------------------------------------ |
| **LLM / Transformer 隐藏层** | GELU                             | 现代大模型的标准配置                 |
| **YOLO系列骨干网络 (CV)**    | SiLU (Swish)                     | YOLOv5及之后版本的默认激活函数       |
| **经典CNN / 密集全连接层**   | ReLU / LeakyReLU                 | 算力成本低，baseline的首选           |
| **二分类任务输出层**         | Sigmoid                          | 将单路输出转化为 $0 \sim 1$ 的概率值 |
| **多分类任务输出层**         | Softmax                          | 将多路logits转化为概率分布           |
| **循环神经网络 (RNN/LSTM)**  | Tanh (状态更新) / Sigmoid (门控) | 传统序列模型的标配结构               |

# 企业项目常见搭配

> 💡 **PyTorch 避坑提示**：在实际开发中，如果多分类网络最后一层使用 `nn.CrossEntropyLoss()`，或者二分类使用 `nn.BCEWithLogitsLoss()`，**网络输出层不需要显式添加 Softmax 或 Sigmoid**。因为这两个损失函数在 PyTorch 底层已经隐式集成了激活计算，直接传入原始的 Logits 能够获得更高的数值稳定性！

| **项目类型**                   | **网络层类型**            | **激活函数搭配**                   | **对应损失函数**  |
| ------------------------------ | ------------------------- | ---------------------------------- | ----------------- |
| **大语言模型 (如类GPT/LLaMA)** | Transformer Block         | GELU / SwiGLU                      | CrossEntropyLoss  |
| **现代目标检测 (YOLOv8)**      | Backbone / Head           | SiLU (Swish)                       | CIoU Loss + BCE   |
| **图像分类 (经典ResNet)**      | Convolution / Linear      | ReLU                               | CrossEntropyLoss  |
| **文本情感分析 (双分类)**      | Output Layer              | Sigmoid (或直接输出Logits)         | BCEWithLogitsLoss |
| **生成对抗网络 (GAN)**         | Generator / Discriminator | Tanh (输出层) / LeakyReLU (隐藏层) | BCELoss           |

# 学习优先级（必须掌握）

| **优先级** | **激活函数**     | **掌握程度**                                                 |
| ---------- | ---------------- | ------------------------------------------------------------ |
| ⭐⭐⭐⭐⭐      | **ReLU**         | **必须掌握**（深度学习基石，必须理解其死区问题,即神经元死亡,梯度=0） |
| ⭐⭐⭐⭐⭐      | **Softmax**      | **必须掌握**（多分类的核心，需理解其如何转化为概率）         |
| ⭐⭐⭐⭐⭐      | **Sigmoid**      | **必须掌握**（二分类及概率映射的基础）                       |
| ⭐⭐⭐⭐       | **GELU**         | **建议掌握**（现代NLP、Transformer大模型的默认标配）         |
| ⭐⭐⭐⭐       | **SiLU (Swish)** | **建议掌握**（现代高效CV模型、YOLO系列的前沿标配）           |
| ⭐⭐⭐        | **LeakyReLU**    | **理解即可**（常用于GAN或缓解ReLU坏死的平替方案）            |
| ⭐⭐⭐        | **Tanh**         | **了解即可**（理解传统RNN或特定生成任务时有用）              |

# 一张图记忆

Plaintext

```
处理隐藏层激活
    │
    ├── 是现代大模型(Transformer/LLM)?  ──→ GELU
    │
    ├── 是现代CV模型(YOLOv5+/EfficientNet)? ──→ SiLU (Swish)
    │
    └── 是经典CNN/全连接层网络? 
             │
             └── 是否遇到神经元坏死(Dead ReLU)?
                      ├── 是 ──→ LeakyReLU
                      └── 否 ──→ ReLU

处理最后一层输出
    │
    ├── 二分类任务? ──→ Sigmoid (或结合BCEWithLogitsLoss不加)
    │
    └── 多分类任务? ──→ Softmax (或结合CrossEntropyLoss不加)
```
