# AI 学习路线

这条路线以“先建立整体认知，再理解训练过程，最后进入大模型应用”为主线。

## 第一阶段：深度学习基础

目标：理解模型如何从数据中学习。

- 神经网络、参数、前向传播与反向传播
- 激活函数、损失函数和优化器
- 训练集、验证集、测试集
- 过拟合、正则化和模型评估

建议先阅读《深度学习核心讲义：通俗版》，再阅读各专题笔记。

## 第二阶段：PyTorch 实践

目标：能够独立写出完整训练循环。

- 张量与 Shape 变化
- Dataset 与 DataLoader
- 模型定义与参数初始化
- 训练、验证、保存和加载模型

学习时建议自己实现一个小型分类任务，并记录每一层输入输出的 Shape。

## 第三阶段：序列模型与 Transformer

目标：理解现代大语言模型的基础结构。

- RNN 的信息传递方式与局限
- Attention 的核心思想
- Transformer 编码器与解码器
- 大语言模型的训练与推理概念

## 第四阶段：大模型应用开发

目标：能够构建安全、可复现的最小应用。

- Prompt 设计
- OpenAI 兼容接口调用
- 流式输出
- RAG、工具调用与工作流

## 学习方法

1. 每学一个概念，尝试用自己的话重新解释。
2. 每个代码示例至少亲手运行并修改一次。
3. 记录报错原因和解决过程。
4. 优先理解输入、输出和 Shape，再关注复杂实现。
5. 定期回顾并修正旧笔记。
